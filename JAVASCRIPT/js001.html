산술연산자

  • 더하기 : +
  • 빼기 : -
  • 곱하기 : *
  • 나누기 : /
  • 나머지 : % (나누는 수의 나머지값)

작성법
변수로 숫자를 지정한다
let 또는 var 또는 const 로 변수를 말한다. (선언문)
let : 가장 유연하게 사용할수있고, 모든 변수이름 앞에 사용 할 수 있다. 에러가 나지않음
var : 처음으로 생긴 변수 지정법으로 가장 오래된 역시를 가지고 있으며 많은 오픈파일에서 볼 수 있다. 단점은 중복되더라도 발견하지못한다.
const : 고유명사 같은 개념으로 변수를 바꿀 수 없다. 대부분 선택자의 이름으로 한번 할당되면 바뀌지않는 값에 쓰인다.
let a = 10
풀이 : a는 10이다.
let b = 5
풀이 : b는 5이다
let c ;
풀이 : C를 변수 이름으로 사용하겠다. c- c=5 + 10. 풀이 : C는 5+10이다.

비교연산자

변수가 두가지가 필요하며 2개의 데이터를 보등호를 통해비교하며 바르면 true,
틀리면 false가 풀력된다.

  • A가 B보다 크다 : A>B
  • A가 B보다 크거나 같다 : A>=B
  • A가 B와 같다 : A==B
  • A가 B와 다르다 : A!=B
  • A가 50보다 크거나 20보다 작다 : A>50 && A<20

논리연산자

  • (그리고) and : &&
  • (또한) or : //
  • (not) 아니다 : !

예제풀이

나이를 물은 후 20세 이상이면 true, 이하이면 false 가 나타나게해라
만약에 age가 20보다 크면
'당신은 성인입니다.'
라는 말을 document에 출력하고
그렇지 않으면
'당신은 성인이 아닙니다.' 라고 출력해라.

증감연산자

변수에 지정된 값이 1씩 증가하거나 감소한다

  • 증가 : ++
  • 감소 : --

대입연산자

연산된 데이터를 지정된 변수에 대입한다.

  • = 은 뒤에 있는 숫자를 내용을 앞의 변수에 대입한다.
  • 변수 = 5
    풀이 : 5는 변수다 변수 +=5
    풀이 : 5를 더해서 변수에 대입해라
