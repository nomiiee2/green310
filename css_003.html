CSS 레이아웃

CSS를 활용하여 공간을 나누듯 배치를 하기 위해 여러 속성을 사용한다.

가시속성

display : 태그가 눈에 보이는 방식을 지정한다.

  • none - 화면에서 안보이며 공간도 함께 사라진다.
    웹접근성으로도 확인할 수 없다.(삭제기능)
  • inline - 특별한 공간의 값을 가지고 있지 않다.
    width, height등의 값을 지정할 수 없다.
    margin, padding의 경우 요소를 기준으로 하기 때문에 좌우는 적용이 되나, 상하는 적용되지않는다.
  • inline-block - inline과 block의 중간역할 두 속성의 모든 성질을 가지고 있음
  • block - 가로값 화면 기준 100%로 적용
    자동 줄바꿈 기능이 있다.

flex 속성

태그를 수평, 수직으로 자유롭고 유연하게 변동할 수 있는 기능

  • container - 부모요소를 부르는 명칭으로 사용함
  • item - 자식요소를 부르는 명칭

Contanier 속성에는 display, flex-flow(묶음기능), flex-derection(수평,수직)
justify-content(아이템 수평정렬), aline-item(아이템 수직정렬)

item 속성에는 flex, grow(증가너비), shrink(감소너비), basis(크기배분), aline-self(각자조정)

자손자에만 오롯이 적용된다. 후손자는 적용안됨(상속되지 않음)

position - 요소를 자유롭게 배치레이아웃

여러요소를 순서와 상관없이 화면에 가로 세로 좌표값을 기준으로 배치
특정영역안에서 위치조절을 자유롭게 할 수 있어 유연한 레이아웃이 가능

  • relative - 적용된 요소의 현재 위치값을 기준으로 상대 위치를 지정한다.
    (기준값 - 부모요소에 주되게 사용 또는 z-index 값을 적용할 시 사용)
  • absolute - 부모요소의 특정 구간을 기준으로 절대 위치를 지정한다.
    (움직이는 요소)
    부모요소에 relative를 지정하지 않을 경우 윈도우를 기준으로 움직인다
  • fixed - 웹브라우저를 기준으로 절대위치를 지정한다. (화면기준으로 보임)
