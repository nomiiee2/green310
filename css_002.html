박스속성

웹페이지 레이아웃시 가장 중요한 속성

  • 요소크기 설정 - width, height
  • 영역설정 - padding
  • 외곽선설정 - border
  • 여백설정 - margin

작성법

  • 전체적용 - 숫자와 단위를 함께 하나만 작성
  • 각자적용 - 숫자와 단위를 4번써서 작성한다.
    순서는 위, 오른쪽, 아래, 왼쪽(시계방향) 순서
    작성시 쉼표를 쓰지않고 띄어쓰기를 통해 작성한다
  • 세트적용 - 위아래/왼쪽오른쪽 숫자와 단위를 2번써서 작성한다
    앞이 상하, 뒤가 왼오 각자적용과 마찬가지로 띄어쓰기를 통해 작성한다
  • 부분적용
    • marjin-top : 위쪽여백
    • marjin-botton : 아래쪽여백
    • marjin-left: 왼쪽여백
    • marjin-right : 오른쪽여백
    • padding-top : 위쪽영역
    • padding-botton : 아래쪽영역
    • padding-left : 왼쪽영역
    • padding-right : 아래쪽영역

border설정방법

박스의 테두리를 말하며, padding바깥쪽에 위치한다.
색상, 굵기, 모양으로 설정한다.

  • border-color : 색상기재
  • border-width : 숫자와 단위
  • border-style
    • none - 선을없앰
    • solid - 실선
    • dotted - 둥근점선(간격설정안됨)
    • dashed - 긴 사각형 점선
    • double - 실선두줄
    • groove - (액자형) - 테두리가 파인것처럼 그려짐
    • ridge - (액자형) - 테두리가 튀어나온것처럼 그려짐 - groove의 반대 스타일
    • inset - (액자형) - 내부요소가 파인것처럼 그려짐
    • outset - (액자형) - 내부요소가 튀어나온것처럼 그려짐 - inset의 반대 스타일

작성법 - color,width,style을 순번에 상관없이 한번에 쓸수도 있고 각자쓸수도 있다.
border : red 2px solid (2px너비의 빨간색실선)
